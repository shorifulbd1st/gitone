* mkdir (folder name) — open new folder
* cd folderName – into the folder
* cd ../(out of folder)
* touch fileName.ext—open new file
* pwd folderPath
* ls folderStatus
* git add .(add all file to stage)
* git reset(remove stage)
* git add * (delete file not up to stage)
* git reset --hard (delete file again add)
* git rm two.txt -f(file delete)
* git rm -r folderName(remove recursive folder)
* git branch branchName(create a new branch)
* git checkout branchName(switched to branch)
* git touch(create new file) then git add. git commit -m"message" now if you switched main branch (remove development branch file)
